Building PeptidePin (peptidepin.com) - a peptide dosage calculator and tracking app.

**Why:** Existing tools (PepDose.com, PepCalc) lack accounts, dose tracking, and reminders. PeptidePin fills the gap.

**How to apply:** Project lives at C:\Users\idjar\peptidepin (renamed from "dosage calculator" due to npm naming). Stack: Next.js 15 App Router, Supabase Auth + PostgreSQL, Tailwind v4 + shadcn/ui, Serwist PWA, Framer Motion. Deploy to Vercel. Domain: peptidepin.com.
