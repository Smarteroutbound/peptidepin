## TECHNICAL SEO

### Schema Markup Per Page
- `WebApplication` — for calculator tools
- `FAQPage` — rich snippets in Google

### PWA SEO (Next.js Advantage)
- SSR/SSG ensures full HTML on first crawl
- Clean URLs: /calculator/semaglutide (not hash-based)
- Dynamic XML sitemap for all calculator pages
- Unique meta tags + OG tags per page
- Core Web Vitals: LCP <2s, INP <200ms, CLS near 0
